# Chemistry & Data Workspace (P1)

Welcome to the Chemistry & Data Lead workspace! 

## Your Tasks:
1. **Curate Greenness Data**: Gather and format toxicity metrics, E-factors, biodegradability, and VOC parameters for candidate solvents.
2. **Define Metric Formulations**: Finalize formulas for the Green Solvent Score and document standard normalization coefficients.
3. **Write the Scientific Note**: Detail validation standards, bias, and chemical safety protocols.
4. **Prepare validation reactions**: Assemble a set of 5 standard reactions with realistic yield profiles and expected green solvent choices to run integration sanity checks.

## Recommended Folder Layout:
- `raw_data/`: Place raw ECHA, PubChem, and literature tables here.
- `processed_data/`: Save final cleaned datasets as CSV or JSON matching the database schema.
- `docs/`: Put drafts of the Scientific Note and formulation documents here.
